# Node.js MySQL Task API

A professional RESTful API built with Node.js, Express, and MySQL.

## 🛠️ Setup
1. Run `npm install`.
2. Import the database schema into MySQL.
3. Configure the `.env` file with your local credentials.
4. Run `node server.js`.

## 🔗 Endpoints
- `GET /tasks` - Fetches all tasks from the MySQL database.
